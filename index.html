New Blog Site With Nanoc

January 25, 2016

Spent a weekend to make this shinny new blog site with Nanoc and some other open source tools.

Summary:

  • Nanoc as static site generator

  • Lanyon as the theme you are seeing right now
  • Guard for regenerating site and live-reloading browser
  • Rake for manging tasks
  • Pygments for syntax highlighting
